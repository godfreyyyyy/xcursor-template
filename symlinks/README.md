Symlinks may be provided for cursors.
They should have the same name as cursors in `src` directory, but with `.links`
file extension.

Every symlink must be placed on the new line.

Example of symlinks for arrow:
```
left_ptr
```

And for pointer:
```
9d800788f1b08800ae810202380a0822
e29285e634086352946a0e7090d73106
hand1
hand2
pointing_hand
```

For more information visit
[KDE Classic Cursors](https://github.com/na0miluv/kde-classic-cursors/)
repository. In file
[`cursorList`](https://github.com/na0miluv/kde-classic-cursors/blob/main/src/cursorList)
you can find all needed symlinks for cursors (left &mdash; symlink name,
right &mdash; file it links to).
